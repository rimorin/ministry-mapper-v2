# 릴리스 노트 (한국어)

## 2026-05-05

[NEW] 스마트 동기화.
  신호가 약해도 업데이트가 즉시 저장됩니다. 방문을 저장하려면 항상 인터넷 연결이 필요했습니다 — 엘리베이터, 지하 주차장, 농촌 지역, 또는 모바일 데이터가 불안정한 곳에서는 그게 쉽지 않았습니다.

  저장을 누르는 순간, 업데이트는 기기에 저장되고 지도에 즉시 반영됩니다. 연결이 준비되면 시스템이 백그라운드에서 서버로 전송합니다.

  - 구역 제목 옆의 📤 배지는 아직 전송 중인 업데이트 수를 보여줍니다. 모두 완료되면 사라집니다.
  - 아직 동기화되지 않은 주소에는 빨간 점이 표시됩니다.

  평소처럼 방문을 기록하세요 — 스마트 동기화가 나머지를 처리합니다.

## 2026-04-23

> 공개 스토리나 상태에 지도 링크를 올리지 마시기 바랍니다 (예: WhatsApp 상태 등) — 회중 밖의 사람들에게도 보일 수 있습니다.

[ANNOUNCEMENT] 메시지 앱에서 지도 링크를 공유할 때 부디 주의해 주세요.
  지도 링크는 전도인이 로그인 없이 바로 지도로 이동할 수 있게 해줍니다 — 즉 링크를 받은 누구든, 외부인을 포함하여 접근할 수 있습니다. 실수로 공개적으로 게시했다면 즉시 해당 게시물을 삭제하고, 담당 인도자나 관리자에게 알려 링크를 삭제하고 새 링크를 보내도록 요청해 주세요.

[ANNOUNCEMENT] 관리자: 링크 만료 기간 단축을 부디 고려해 주세요.
  더 짧은 "수명"을 설정하면 링크가 잘못된 곳에 공유되더라도 빠르게 만료되어 위험 시간이 매우 짧아집니다.

## 2026-04-11

[NEW] 전도인이 추가한 새 주거 지역 주소에 대한 일일 요약 이메일.
  관리자는 지난 24시간 동안 추가된 주소의 일일 요약을 받습니다. 각 항목에는 추가한 사람과 방문 내용이 표시되어 기록을 쉽게 검토하고 정확하게 유지할 수 있습니다.

> 관리자: 이메일 알림이 켜져 있는지, Ministry Mapper 이메일이 스팸함으로 이동하지 않는지 확인해 주세요.

## 2026-04-08

[NEW] 현장에서 바로 주소 추가.
  이제 전도인이 주거 지역 링크를 작업하는 중에 누락된 주소를 직접 추가할 수 있어, 구역 담당자와 봉사 감독의 조사 부담이 줄어듭니다.

  이 기능은 아직 구역 기록을 구축 중이거나 전혀 없는 회중을 위해 만들어졌습니다. 미등록 구역이 있다면 실제 전도 활동을 통해 채워 나갈 수 있습니다 — 구역은 실제 전도 작업을 통해 스스로 문서화됩니다.

  사용 방법:
  - 주소 목록 맨 끝의 + 카드를 탭합니다.
  - 건물 번호와 방문 내용을 입력한 후 만들기를 탭합니다.

  새 주소는 즉시 저장되며, 같은 구역에서 작업 중인 모든 사람에게 실시간으로 표시됩니다.

> 새 주소는 목록 맨 끝에 추가됩니다. 관리자만 목록을 재정렬할 수 있습니다.

[ANNOUNCEMENT] Ministry Mapper는 웹 브라우저에서 열어야 합니다.
  홈 화면에 저장하거나 설치된 앱으로 사용하는 것은 지원되지 않습니다 — 이미 그렇게 하셨다면 제거하고 브라우저에서 링크를 직접 여세요.
